# 🚀 Mini Projeto 1: 💸 PyFinanceiro

O PyFinanceiro é um projeto desenvolvido em Python para realizar análises financeiras a partir de registros armazenados em um arquivo CSV.

## 🛠 Tecnologias

O projeto foi construído aplicando recursos do:

- Python (3.13.13)

## 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina o [Git](https://git-scm.com) e o Python.

## 📂 Estrutura dos Dados

Os dados financeiros devem estar armazenados em um arquivo CSV. O script realiza a leitura desse arquivo e executa as análises automaticamente.

Exemplo:

Data,Lucros/Perdas

Jan-2010,867884

Feb-2010,984655

## 📊 Funcionalidades
Leitura de dados financeiros em formato CSV.

Cálculo de receitas e despesas.

Análise do saldo financeiro com geração de um arquivo .txt.
